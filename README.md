# ESP32-Smart-Relay-Control-using-Blynk-IoT

Control electrical devices remotely using ESP32 and Blynk Cloud. This project allows you to turn a relay ON/OFF from anywhere using a smartphone.

🚀 Project Overview

This IoT project connects an ESP32 to the Blynk Cloud.
Using a button widget in the Blynk mobile app, a relay module can be controlled remotely.

When the button is pressed:

Relay turns ON

Device connected to relay activates

When button is OFF:

Relay turns OFF

⚙ How It Works

ESP32 connects to WiFi

ESP32 connects to Blynk Cloud

Button press sends value to Virtual Pin V1

ESP32 receives value

Relay switches ON/OFF
